class Backend a where

The functions every backend window managed backend needs to support.

The Backend module interfaces with the window manager, and handles opening and closing the window, and managing key events etc.

It doesn't know anything about drawing lines or setting colors. When we get a display callback, Gloss will perform OpenGL actions, and the backend needs to have OpenGL in a state where it's able to accept them.
